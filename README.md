# Residuum (Frontend)

Scaffold inicial do frontend do Residuum (React + Vite + Tailwind + TanStack Query + Axios + React Hook Form + Zod)

Scripts:

- `npm run dev` — inicia o dev server (Vite)
- `npm run build` — gera build de produção
- `npm run preview` — preview do build
- `npm run format` — formata com Prettier

Próximos passos:
- `npm install`
- configurar variáveis `VITE_API_BASE_URL` e `VITE_GOOGLE_MAPS_KEY` em `.env`
